In this scheme, a special block known as the Index block contains the pointers to all the blocks occupied by a file. Each file has its own index block. The ith entry in the index block contains the disk address of the ith file block. The directory entry contains the address of the index block as shown in the below example.

  • Advantages
    • This supports direct access to the blocks occupied by the file and therefore provides fast access to the file blocks.
    • It overcomes the problem of external fragmentation.
  • Disadvantages
    • The pointer overhead for indexed allocation is greater than linked allocation.
